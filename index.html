Day 1: Introduction to HTML + Structure and Text

What is HTML and why is it used?

HTML (HyperText Markup Language) is the standard language used to create webpages.

It defines the structure of content (headings, paragraphs, links, etc.).

Setting up VS Code for web development

  • Download and install Visual Studio Code.
  • Install the Live Server extension for real-time preview.

Creating your first HTML file

  • Create a new file named index.html.
  • Save it in a project folder.

Understanding HTML document structure:

  • <!DOCTYPE html>: Declares the document type.
  • <html>: Root of the document.
  • <head>: Metadata like title, styles, etc.
  • <title>: Page title in the browser tab.
  • <body>: Main content visible to users.
<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>

Text Elements

  • Headings: <h1> to <h6> define different sizes.
  • Paragraphs: <p> to separate blocks of text.
  • Line Breaks: <br>
  • Inline text: <span>, <code>, <pre>
  • Hyperlinks: <a href="">, target="_blank"
  • Comments: <!-- comment -->

Day 2: Lists, Media, and Attributes

HTML Lists

  • Ordered List: <ol>
  • Unordered List: <ul>
  • List Items: <li>
  • Nesting Lists: Combining <ul> or <ol> inside <li>
<ul>
    <li>Item 1</li>
    <li>Item 2
        <ol>
            <li>Nested item 1</li>
            <li>Nested item 2</li>
        </ol>
    </li>
</ul>

Media Tags

  • <img src="">, alt, width, height
  • <video> and <audio>: src, controls, autoplay, loop, muted
  • <source> tag for alternate file formats
  • <iframe>: Embed another webpage or media
<img src="image.jpg" alt="Description" width="300" height="200">

<video controls width="400">
    <source src="video.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

HTML Attributes Overview

  • Global attributes: id, class, style
  • Media and link attributes: href, target, src, alt, width, height

Day 3: Divs, Tables, and Forms

Div and Formatting Elements

  • <div>: Block-level container for layout
  • Formatting: <b>, <i>, <u>, <strong>, <em>, <mark>
<div class="container">
    <p>This is <b>bold</b> and this is <i>italic</i>.</p>
    <p>This is <strong>important</strong> and this is <em>emphasized</em>.</p>
</div>

HTML Tables

  • <table>: Main container
  • <tr>, <td>, <th>: Rows, data, headers
<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>Country</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
        <td>USA</td>
    </tr>
</table>

Forms and Input Types

  • <form>, <label>, <input>, <textarea>, <select>, <option>, <button>
  • Input types: text, email, checkbox, radio, color, range, date, number, file, submit
<form>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <button type="submit">Submit</button>
</form>

Day 4: CSS Basics and Styling

What is CSS and Why It Matters

CSS (Cascading Style Sheets) controls how HTML looks on the screen.

Ways to Add CSS

  • Inline: <element style="">
  • Internal: <style> inside <head>
  • External: Linked .css file with <link>
<!-- External CSS -->
<link rel="stylesheet" href="styles.css">

<!-- Internal CSS -->
<style>
    p { color: blue; }
</style>

<!-- Inline CSS -->
<p style="color: red;">This is red text</p>

Selectors and Specificity

  • Element: p {}
  • Class: .classname {}
  • ID: #idname {}
  • Specificity order: Inline > ID > Class > Element

Text Styling

  • font-family, font-size, font-weight, text-align, text-decoration, text-transform, line-height, letter-spacing, word-spacing, text-shadow

Colors and Units

  • Colors: name, #hex, rgb(), rgba(), hsl(), hsla()
  • Units: px, em, rem, %, vh, vw
